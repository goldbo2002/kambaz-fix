
# Kambaz Dashboard


A canvas mimic

# Tech

front: React, Vite, Redux Toolkit, TypeScript
back: Node.js, Express, Mongoose
database: MongoDB Atlas
authentication: Session-based with express-session

# demo

https://kambaznew.netlify.app/
---


# Backend

```bash
cd server
npm install
node index.js
```

# Frontend

```bash
cd client
npm install
npm run dev
```
